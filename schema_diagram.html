⚡ Cambios clave v1.0 → v2.0

ADD Tablas grupos y razones_sociales: jerarquía multi-tenant completa
ADD Tabla configuracion_visibilidad: visibilidad cruzada entre RS configurable por grupo
CHG sucursales: ahora dependen de razon_social_id, campo codigo (MB1, H2…)
CHG clientes: sucursal_id → grupo_id. UNIQUE cambia a (grupo_id, whatsapp). Un cliente por grupo, sin duplicados.
CHG empresas: sucursal_id → grupo_id. Empresas únicas a nivel grupo.
ADD actividades: campos entidad_tipo + entidad_id para lookup universal cross-módulo en reportes.
CHG RLS: políticas actualizadas con helpers get_mi_grupo_id(), get_mi_rs_id(), get_mi_sucursal_id().
ADD Índice idx_citas_limite (parcial, solo pendiente_contactar) para optimizar el timer 15 min.
